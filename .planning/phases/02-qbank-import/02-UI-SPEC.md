# Phase 2 — UI Design Contract

> Visual and interaction contract for **题库导入**（单选 / 多选 / 判断批量导入、校验反馈、抽样预览与列表浏览）。Generated by gsd-ui-researcher; verify with gsd-ui-checker before planning execution.

**Scope:** 管理端 `/admin/questions`；仪表盘「题库」入口由 disabled 改为可导航；Excel `.xlsx` 上传与官方模板下载；导入成功摘要 + 本批抽样预览；行级错误表；分页题目列表与详情预览 Dialog。中文界面。延续 Phase 1 `AdminLayout`、色板与文案语气。

**Out of scope (UI):** 在线命题编辑器、警种筛选/展示（D-09 列不入库，预览与列表均不展示警种）、名单与考试模块（仪表盘卡片保持 disabled）。

## Phase Surfaces

### S0 — 仪表盘「题库」入口 (`/admin`)

| Element | Spec |
|---------|------|
| 可点击卡片 | `题库` 使用 `react-router-dom` `Link`，`hover:border-primary/50`，`focus-visible:ring-2 ring-ring` |
| 其余卡片 | 不变：`aria-disabled`、`opacity-60` |

### S1 — 题库管理页 (`/admin/questions`)

**Purpose:** QBANK-01～03 统一入口；教师完成模板下载、上传、查看结果与浏览题库。

| Element | Spec |
|---------|------|
| 页面标题 | Heading（20px）：「题库管理」 |
| 说明 | Body（16px）：「请使用官方 Excel 模板批量导入单选、多选与判断题。导入前可下载模板并按「填写说明」填写。」 |
| 返回 | 页标题上方或左侧 Body 链接「← 返回仪表盘」→ `/admin`（14px semibold，accent 色文字 `text-primary`） |

### S2 — 导入区（页内 Card #1）

| Element | Spec |
|---------|------|
| Card 标题 | Label 档（14px semibold）：「导入题库」 |
| 下载模板 | `Button` `variant="outline"`，图标 `Download`；文案见 Copywriting；点击 `GET /api/admin/questions/template`，浏览器下载 `题库导入模板.xlsx` |
| 上传区 | 虚线边框区域，最小高度 **160px**，圆角 `rounded-lg`，`border-2 border-dashed border-border`；居中图标 `Upload` + Body 主文案 + 辅助行「仅支持 .xlsx，单文件不超过 5MB」 |
| 交互 | 点击区域唤起文件选择；支持拖拽 `drop`；选中后显示文件名 + `更换文件` 文字按钮（outline/link） |
| 文件校验（前端） | 扩展名 `.xlsx`；`file.size ≤ 5_242_880`；不合规时 Alert destructive，不发起请求 |
| 主 CTA | 全宽（移动端）或右对齐（`md+`）Primary：「开始导入」；无文件时 `disabled` |
| 提交中 | 主按钮 loading + 上传区 `pointer-events-none opacity-60`；文案「正在导入…」 |

**布局间距：** Card 内 padding 24px（`p-6`）；下载按钮与上传区间距 16px；上传区与主 CTA 间距 24px。

### S3 — 导入成功反馈

**Trigger:** `POST /api/admin/questions/import` 返回 `200` 且 `ok: true`。

| Element | Spec |
|---------|------|
| Toast | `sonner` success：文案见 Copywriting（含导入条数） |
| 摘要 Card | 边框 `border-primary/30`，背景 `bg-card`；Heading 行「导入成功」+ Body 三行统计：`成功导入 {importedCount} 题`、`跳过示例行 {skippedCount} 条`（为 0 时隐藏该行）、`来源文件：{fileName}` |
| 抽样预览 | 摘要 Card 下方子标题 Label：「本批预览（前 3 题）」；最多 3 条紧凑预览卡（见 S4 预览块简化版）；底部 `Button` outline「查看全部题目」→ 平滑滚动至列表区（`scrollIntoView`）或聚焦列表 Card |
| 列表刷新 | 成功后自动 `refetch` 题目列表（重置到第 1 页） |

### S4 — 导入失败反馈（行级错误）

**Trigger:** `400` 且响应含 `errors[]`（整批未写入，无脏数据）。

| Element | Spec |
|---------|------|
| Toast | `sonner` error：文案见 Copywriting |
| Alert | 表单级 destructive Alert 置于错误表上方，复述「未写入任何题目」 |
| 错误表 | shadcn `Table`；列：**行号**（Excel 行，含表头则从 2 起）、**列名**、**原因**；按行号升序；超过 50 行时表体 `max-h-[320px]` 可滚动（`ScrollArea` 或 `overflow-y-auto`） |
| 可选 | 表上方 Body：「共 {n} 处错误，请修正 Excel 后重新导入。」 |

**禁止：** 在失败态展示「部分成功」计数（与事务策略一致）。

### S5 — 题目预览块（抽样 + Dialog 共用内容模型）

**单条预览块（列表行点击 / 抽样卡共用）：**

| 字段 | 展示 |
|------|------|
| 题型 | `Badge`：`单选` / `多选` / `判断`（variant：`secondary`） |
| 题干 | Body 16px，最多 3 行 `line-clamp-3`（列表摘要）；Dialog 内全文换行 |
| 选项 | 列表：`A. …` 单行摘要（最多显示至 C 项 +「…」）；Dialog：逐行 `A`–`F`（仅非空选项），判断题固定 A=正确、B=错误 |
| 答案 | Label 行：「正确答案：」+ 展示规范化答案（多选为 `A、B、C` 顿号分隔） |
| 解析 | Dialog 内若有则 Body muted；无则「—」 |
| 知识点 | Dialog 内若有则 Body；无则隐藏该行 |
| 分值 / 难度 | Dialog 内 Label 行：「分值 {points} · 难度 {difficulty}」 |
| 多选计分 | 仅 `多选`：Dialog 底部 muted 辅助行「计分规则：全对得满分，否则不得分」（对应 `ALL_OR_NOTHING`） |

### S6 — 题目列表（页内 Card #2）

| Element | Spec |
|---------|------|
| Card 标题 | 「题目列表」 |
| 筛选 | 一行内：`Select` 题型（全部 / 单选 / 多选 / 判断）；可选 `Select` 导入批次（默认「全部」；含「最近一次导入」选项 — 取 `batchId` 最新一条） |
| 表格列 | 题型（Badge）｜题干摘要（`line-clamp-2`，flex-1）｜分值｜导入时间（`yyyy-MM-dd HH:mm` 本地）｜操作「查看」 |
| 行交互 | 点击行或「查看」打开 S7 Dialog；行 `hover:bg-muted/50`，`cursor-pointer` |
| 分页 | 每页 **20** 条；底部分页：「上一页」「下一页」+ Body「第 {page} / {totalPages} 页，共 {total} 题」；首页/末页禁用对应按钮 |
| 空状态 | 无题目时：居中 `BookOpen` 图标 + Empty copy（见 Copywriting）；**隐藏** 分页 |

**加载态：** 表格区域 Skeleton 3 行或居中 Spinner +「加载题目列表…」。

### S7 — 题目详情 Dialog

| Element | Spec |
|---------|------|
| 组件 | shadcn `Dialog`；最大宽度 `max-w-lg`；内容区 `max-h-[80vh] overflow-y-auto` |
| 标题 | DialogTitle：题型 Badge + 题干前 40 字（超长省略） |
| 关闭 | 右上角默认关闭；底部 Primary「关闭」 |
| 内容 | 完整 S5 预览字段 |

## Color

继承 `index.css` / Phase 1 契约。

| Role | Value | Phase 2 usage |
|------|-------|---------------|
| Dominant (60%) | `#F8FAFC` (`--background`) | 页面背景 |
| Secondary (30%) | `#FFFFFF` (`--card`) + `#E2E8F0` (`--border`) | 导入 Card、列表 Card、表格、上传虚线框 |
| Accent (10%) | `#2563EB` (`--primary`) | 见下方保留列表 |
| Destructive | `#DC2626` (`--destructive`) | 导入失败 Alert、错误 Toast、校验 Alert |
| Muted text | `#64748B` (`--muted-foreground`) | 辅助说明、空状态、计分规则提示 |

**Accent reserved for:**

1. Primary 按钮（开始导入、Dialog 关闭）
2. 下载模板 outline 按钮 focus ring
3. 返回仪表盘链接文字
4. 成功摘要 Card 左边框强调（`border-l-4 border-l-primary` 可选）
5. 分页当前页不可accent背景 — 使用 `secondary` 即可

**禁止：** accent 用于表格行背景、题型 Badge 背景（Badge 用 `secondary`）。

- 题型 Badge 映射：`SINGLE`→单选，`MULTI`→多选，`JUDGE`→判断

## Interaction & Accessibility

| Rule | Spec |
|------|------|
| 语言 | `html lang="zh-CN"` |
| 上传区 | `role="button"` 或内嵌可见 file input；`aria-label="选择 Excel 文件"`；拖入时 `aria-live="polite"` 播报文件名 |

| 键盘 | Dialog 陷阱焦点；Esc 关闭；上传区 Enter/Space 触发文件选择 |
| 对比度 | 沿用 Phase 1（`foreground` on `background` / `card`） |
| 响应式 | 最小 320px；小屏表格横向 `overflow-x-auto`；导入主按钮小屏全宽 |
| 动效 | 仅按钮 loading / Toast；**无** 页面转场动画 |
