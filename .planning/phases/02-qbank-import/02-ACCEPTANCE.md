## 需求验收（QBANK-01 / 02 / 03）

### QBANK-01 单选题批量导入

- [ ] 模板「单选」行可导入；`type=SINGLE`，答案为单字母（如 `A`）
- [ ] 列表筛选「单选」仅显示 SINGLE 行

### QBANK-02 多选题批量导入与计分规则

**锁定规则（实现与库表一致）：**

| 字段 | 值 | 含义 |
|------|-----|------|
| `Question.multiScoringRule` | `ALL_OR_NOTHING` | 全对满分，否则 0 分 |

**计分定义（Phase 4 阅卷 MUST 读取 `multiScoringRule`，在服务端计分，勿仅信前端文案）：**

- 学员所选选项集合与 `answerKeys`（规范化逗号分隔，如 `A,B,C`）**完全一致** → 得该题 `points`
- **多选、少选、错选**任一情况 → **0 分**
- 不提供部分分

**UI 可见性（本阶段）：**

- [ ] 导入成功摘要含「全对满分，否则 0 分」说明
- [ ] 多选详情 Dialog 显示固定中文计分说明，且 API 返回 `multiScoringRule: "ALL_OR_NOTHING"`
- [ ] 多选答案以标签 `A` `B` `C` 展示，与 `answerKeys` 一致
- [ ] 列表筛选「多选」仅显示 MULTI 行

### QBANK-03 判断题批量导入

- [ ] 模板「判断」行可导入；`type=JUDGE`，答案 `A`/`B` 或 正确/错误
- [ ] 列表筛选「判断」仅显示 JUDGE 行
