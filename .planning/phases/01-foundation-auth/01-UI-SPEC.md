# Phase 1 — UI Design Contract

> Visual and interaction contract for **基础服务与教师认证**. Generated by gsd-ui-researcher; verify with gsd-ui-checker before planning execution.

**Scope:** 教师登录、首次登录改密、管理端仪表盘占位、路由守卫与未授权态。中文界面。学生端本阶段仅保留根路径占位（无交互表单）。

| Property | Value |
|----------|-------|
| Tool | shadcn/ui |
| Preset | `new-york` style, `zinc` base color, `cssVariables: true` — run `npx shadcn@latest init` in Plan 01-01 |

## URL & Layout Shell

| Route | Surface | Auth |
|-------|---------|------|
| `/` | 学生端占位页（静态文案，无表单） | 公开 |
| `/admin/login` | 教师登录 | 公开 |
| `/admin/change-password` | 首次登录强制改密 | 已登录且 `mustChangePassword=true` |
| `/admin` 或 `/admin/dashboard` | 管理端仪表盘占位 | 已登录且已完成改密 |

**Admin shell（本阶段最小）：**

- 全宽居中卡片布局用于登录/改密（无侧边栏）
- 仪表盘占位使用顶栏 + 主内容区：
  - 顶栏高 56px：`局域网考试系统` 产品名（左）+ 当前教师用户名 + `退出登录` 按钮（右）
  - 主内容区最大宽度 960px，水平居中，上下内边距 32px

## Phase Surfaces

### S1 — 教师登录 (`/admin/login`)

**Purpose:** AUTH-01 入口；错误凭据给出明确反馈，不泄露账号是否存在。

| Element | Spec |
|---------|------|
| 视觉锚点 | Display 标题「教师登录」为页面唯一主焦点（居中卡片顶部） |
| 页面标题 | Display：「教师登录」 |
| 副标题 | Body：「局域网考试系统 · 管理端」 |
| 字段 | `用户名`（text）、`密码`（password，可切换显示） |
| 主按钮 | 全宽 Primary，文案见 Copywriting |
| 禁用项 | **不展示**「忘记密码」链接（D-03） |
| 提交中 | 按钮显示 loading spinner，字段 disabled |
| 错误 | 表单顶部 `Alert` destructive 样式，文案见 Copywriting |
| 键盘 | Enter 在密码框聚焦时提交表单 |

**Card:** 宽 400px（`sm` 断点以下 100% 宽，左右边距 16px），内边距 32px，圆角 8px，阴影 `shadow-sm`。

### S2 — 首次登录改密 (`/admin/change-password`)

**Purpose:** D-02 强制改密；完成前不可进入管理功能。

| Element | Spec |
|---------|------|
| 页面标题 | Display：「设置新密码」 |
| 说明 | Body：「首次登录须修改默认密码后方可使用管理功能。」 |
| 字段 | `当前密码`、`新密码`、`确认新密码`（均为 password） |
| 新密码提示 | Label 下方辅助行：「至少 8 位，建议包含字母与数字」 |
| 主按钮 | 全宽 Primary，文案见 Copywriting |
| 校验（前端） | 新密码 ≥8 位；确认一致；不与当前密码相同 |
| 校验（后端错误） | Alert 展示服务端消息（如「当前密码不正确」） |
| 导航守卫 | 无返回/跳过；访问 `/admin/*` 其他路径均重定向回本页 |

### S3 — 管理端仪表盘占位 (`/admin` 或 `/admin/dashboard`)

**Purpose:** ROADMAP 成功标准 #2；为 Phase 2–4 预留壳层。

| Element | Spec |
|---------|------|
| 欢迎标题 | Heading：「欢迎，{displayName}」 |
| 占位说明 | Body empty-state 文案见 Copywriting |
| 功能预告 | 3 个 disabled 卡片（题库、名单、考试），灰色图标 + 「即将开放」标签 — **不可点击** |
| 主操作 | 顶栏 `退出登录` 即可 |

### S4 — 路由守卫与未授权态

| State | Behavior | UI |
|-------|----------|-----|
| `checking` | 应用启动或路由切换时校验 Session | 全屏居中 `Spinner` + Body 文案「正在验证登录状态…」 |
| `unauthenticated` | 访问 `/admin/*`（除 login）无有效 Session | 重定向 `/admin/login?redirect={path}`；登录成功后跳回 `redirect` 或 `/admin` |
| `mustChangePassword` | Session 有效但首登未改密 | 重定向 `/admin/change-password`；改密成功后跳 `/admin` |
| `authenticated` | 正常访问 | 渲染目标页 |
| `sessionExpired` | 管理端 API 返回 401（`auth/me` 等 hydrate 调用除外，见 D-10 `skipAuthRedirect`） | Toast 文案见 Copywriting → 清 Session → 跳转 login |
| `forbidden` | API 返回 403（本阶段少见） | 内联 Alert：「无权访问此功能」 |

**学生端根路径 `/`：** 单段居中 Body 文案「局域网考试系统 — 学生端即将开放」，无按钮、无导航至 `/admin`（教师从书签/文档获知管理端 URL）。

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | 图标与文字间距、inline 元素 |
| sm | 8px | 表单项之间、Alert 内边距 |
| md | 16px | 卡片内字段间距、页面水平边距（移动端） |
| lg | 24px | 卡片内边距（改密页字段组间距） |
| xl | 32px | 登录卡片内边距、主内容区上下 padding |
| 2xl | 48px | 登录页 Logo/标题区与表单区间距 |
| 3xl | 64px | 全屏居中卡片与视口上下留白 |

**Exceptions:**

- 顶栏高度 **56px**（非 8 倍数，固定 shell 尺寸）
- 表单主按钮最小高度 **44px**（触控友好，内边距垂直 12px）

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 16px | 400 (regular) | 1.5 | 说明文字、辅助提示、占位正文 |
| Label | 14px | 600 (semibold) | 1.4 | 表单标签、卡片小标题、「即将开放」 |
| Heading | 20px | 600 (semibold) | 1.2 | 卡片内次级标题、仪表盘欢迎行 |
| Display | 28px | 600 (semibold) | 1.2 | 登录/改密页主标题 |

仅使用以上 **4 档字号** 与 **2 种字重**（400、600）。

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#F8FAFC` (slate-50) | 页面背景、全屏 loading 背景 |
| Secondary (30%) | `#FFFFFF` + border `#E2E8F0` (slate-200) | 卡片、顶栏、输入框背景 |
| Accent (10%) | `#2563EB` (blue-600) | 见下方保留列表 |
| Destructive | `#DC2626` (red-600) | 错误 Alert、destructive 按钮（本阶段仅错误态） |
| Muted text | `#64748B` (slate-500) | 副标题、辅助说明、disabled 卡片 |
| Border focus | `#2563EB` ring 2px | 输入框 focus-visible |

**Accent reserved for（仅此列表，不得扩散）：**

1. 主按钮（登录、保存新密码）背景
2. 输入框 focus ring
3. 顶栏当前用户名旁在线状态小圆点（可选，8px）
4. 链接文字（本阶段无外链，预留）

**禁止：** 将 accent 用于大面积背景、正文色或装饰性边框。

## Component Inventory (shadcn)

Install during implementation; all from **shadcn official** registry:

| Component | Surface |
|-----------|---------|
| `button` | 登录、改密、退出 |
| `input` | 用户名、密码字段 |
| `label` | 表单标签 |
| `card` | 登录/改密容器、仪表盘占位卡片 |
| `alert` | 表单级错误 |
| `form` | 登录、改密（+ `react-hook-form` + `zod`） |
| `sonner` (toast) | Session 过期提示 |
| `spinner` 或 `loader-2` icon 动画 | 路由 checking、按钮 loading |

**Patterns:**

- 表单错误：字段级用 `aria-invalid` + Label 下方 14px 红色辅助文字；表单级用顶部 Alert
- 密码框：右侧 `lucide-react` `Eye` / `EyeOff` 图标按钮切换可见性，44×44 点击区域；`aria-label` 为「显示密码」/「隐藏密码」（随状态切换）
- 禁用预告卡片：`opacity-60`，`cursor-not-allowed`，无 hover 效果

## Copywriting Contract

| Element | Copy |
|---------|------|
| 产品名 | 局域网考试系统 |
| Primary CTA（登录） | 登录管理端 |
| Primary CTA（改密） | 保存新密码 |
| 退出登录 | 退出登录 |
| Empty state heading（仪表盘） | 管理功能筹备中 |
| Empty state body | 题库导入、名单管理与考试编排将在后续版本开放。当前阶段请确认您可正常登录并访问本页。 |
| Error state（凭据错误） | 用户名或密码错误，请检查后重试。 |
| Error state（改密-当前密码错） | 当前密码不正确，请重新输入。 |
| Error state（改密-不一致） | 两次输入的新密码不一致。 |
| Error state（改密-过短） | 新密码至少需要 8 个字符。 |
| Error state（网络/服务） | 无法连接服务器，请检查网络或联系机房管理员。 |
| Error state（Session 过期） | 登录已过期，请重新登录。 |

| 学生端占位 | 局域网考试系统 — 学生端即将开放 |
| Destructive confirmation | 本阶段无破坏性业务操作；`退出登录` **无需**二次确认，点击即退出并跳转 `/admin/login` |

**语气：** 简洁、中性、不指责用户；不提示「用户名存在但密码错」等账号枚举信息。

## Interaction & Accessibility

| Rule | Spec |
|------|------|
| 语言 | `html lang="zh-CN"` |
| Focus | 所有可交互元素可见 focus ring（accent 色） |
| 表单 | 每项 `label` 关联 `htmlFor`；错误时 `aria-describedby` 指向错误文案 |
| 对比度 | 正文与背景 ≥ 4.5:1（slate-900 `#0F172A` on white/slate-50） |
| 响应式 | 最小支持宽度 320px；登录卡片 `max-w-[400px]` |
| 动效 | 按钮 loading 仅 opacity/spinner；**无**页面转场动画（机房弱机友好） |
