## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 教师浏览器 → multipart xlsx | 不可信文件；仅服务端 ExcelJS 解析 |
| 教师 Session → `/api/admin/roster/*` | 须已登录且完成改密（requireAdminSession） |
| 名单库 nationalId 明文 | 内网机房假设（D-02）；教师端列表默认脱敏展示 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | E | POST /api/admin/roster/import 无会话 | mitigate | `requireAdminSession` on all roster admin routes |
| T-03-02 | D | 超大 xlsx / 行数炸弹 | mitigate | 5MB multipart limit + MAX_ROSTER_IMPORT_ROWS 2000 + ROSTER_IMPORT_RATE_LIMIT_MAX |
| T-03-03 | T | MIME/扩展名伪造 | mitigate | `assertValidXlsxUpload` 扩展名+mimetype+ZIP 魔数 |
| T-03-04 | T | 部分写入脏名单 | mitigate | validate 阶段 errors.length>0 时零 `prisma.create`（ALL_OR_NOTHING） |
| T-03-05 | I | 400 响应泄露堆栈 | mitigate | 仅 `{ row, column?, message }` 行级错误 |
| T-03-06 | I | 日志记录完整证号 | mitigate | import 失败/成功日志不含 nationalId 全文 |
</threat_model>
