<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 学生浏览器 → `/api/student/exam/*` | 不可信 body；`rosterEntryId` 仅来自 `student_sid` |
| 学生 → 试卷 JSON | 不得包含 `answerKeys` |
| 教师浏览器 → `/api/admin/exams/*` | 须 `requireAdminSession`；考试 `teacherId` 归属校验 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigate |
|-----------|----------|-----------|-------------|----------|
| T-04-01-01 | Elevation | exam-paper / exam-answers | mitigate | `assertStudentExamAccess` 校验 `entry.batchId === exam.rosterBatchId` |
| T-04-01-02 | Tampering | exam-answers body | mitigate | 校验 `examQuestionId` 属于该 `examId`；禁止 body 指定他人 `rosterEntryId` |
| T-04-01-03 | Information disclosure | exam-paper | mitigate | 响应 strip `answerKeys`；403 笼统文案 |
| T-04-01-04 | Tampering | POST start | mitigate | 仅 `DRAFT`；教师 session；同 batch 单 IN_PROGRESS |
| T-04-01-05 | Denial of service | PUT answers | mitigate | rateLimit 60/min；zod 限制 `answers` 数组长度 ≤ 题目数 |

</threat_model>
