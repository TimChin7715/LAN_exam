# Roadmap: 局域网考试系统

## Overview

先搭起可在局域网访问的 Web 服务与教师认证，再完成三种题型的导入与存储，继而打通名单与学生强绑定入场，最后实现考试编排、作答提交与成绩/明细导出，形成完整 v1 闭环。

## Phases

- [ ] **Phase 1: 基础服务与教师认证** — 可部署、可访问、教师可登录
- [ ] **Phase 2: 题库导入** — 单选 / 多选 / 判断批量入库
- [ ] **Phase 3: 名单与学生入场** — 名单导入与姓名+身份证强校验
- [ ] **Phase 4: 考试、提交与导出** — 组卷考试、提交约束、成绩与明细导出

## Phase Details

### Phase 1: 基础服务与教师认证
**Goal:** 专用服务器上运行 Web 应用，局域网内可访问；教师可登录管理端，未授权不可访问管理功能。  
**Mode:** mvp  
**UI hint**: yes  
**Depends on:** 无  
**Requirements:** INFRA-01, AUTH-01  
**Success Criteria** (what must be TRUE):

1. 在目标部署方式下启动服务后，从局域网内另一台机器浏览器打开首页或登录页无连接错误
2. 教师使用正确凭据可进入管理端首页或仪表盘占位页
3. 错误凭据或非登录态无法访问需要教师权限的 API 或页面（以验收清单为准）

**Plans:** 3 plans

Plans:

- [ ] 01-01: 技术栈选型、项目骨架与本地/局域网运行说明
- [ ] 01-02: 数据持久化与基础配置（数据库或等价层）
- [ ] 01-03: 教师认证（登录、会话、管理端路由守卫）

### Phase 2: 题库导入
**Goal:** 教师可将单选、多选、判断题通过约定格式批量导入系统并可在管理端校验条数与抽样预览。  
**Mode:** mvp  
**Depends on:** Phase 1  
**Requirements:** QBANK-01, QBANK-02, QBANK-03  
**Success Criteria** (what must be TRUE):

1. 导入合法样例文件后，三种题型在系统中均有记录且可列表查看
2. 导入非法或缺字段文件时，有明确错误提示且不产生半条脏数据（或事务回滚）
3. 单题关键字段（题干、选项、答案）在预览中与源文件一致

**Plans:** 3 plans

Plans:

- [ ] 02-01: 题目数据模型与导入管道（解析、校验、持久化）
- [ ] 02-02: 单选与判断题导入与预览 UI
- [ ] 02-03: 多选题导入、计分规则固定实现与预览

### Phase 3: 名单与学生入场
**Goal:** 教师导入姓名+身份证号名单；学生端输入相同字段后仅匹配成功者进入考试准备态。  
**Mode:** mvp  
**Depends on:** Phase 2  
**Requirements:** ROST-01, AUTH-02  
**Success Criteria** (what must be TRUE):

1. 导入名单后，管理端可按姓名或证件号检索到记录
2. 与名单完全一致的学生凭据可通过校验；任一字段不匹配则拒绝并提示
3. 拒绝原因不泄露其他考生隐私信息（如不提示「证件号对但姓名错」类组合信息，具体文案在实现中统一）

**Plans:** 2 plans

Plans:

- [ ] 03-01: 名单数据模型、导入与检索
- [ ] 03-02: 学生端登录页与强绑定校验 API

### Phase 4: 考试、提交与导出
**Goal:** 教师为考试关联题目与参考名单，学生完成作答并提交；教师导出成绩汇总与答题明细。  
**Mode:** mvp  
**Depends on:** Phase 3  
**Requirements:** EXAM-01, EXAM-02, EXPR-01, EXPR-02  
**Success Criteria** (what must be TRUE):

1. 教师创建一次考试并关联题目与名单后，名单内学生可见试卷并答题
2. 学生提交后，管理端可见最终提交记录；重复提交行为符合 EXAM-02 策略
3. 导出文件包含成绩汇总与题目级明细，可用表格软件打开且编码正确

**Plans:** 3 plans

Plans:

- [ ] 04-01: 考试实体、组卷、学生拉题与作答保存
- [ ] 04-02: 提交、分数计算与防重复提交
- [ ] 04-03: 成绩与明细导出（CSV 或 Excel 择一为主交付）
