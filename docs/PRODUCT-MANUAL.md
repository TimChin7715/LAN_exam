# 局域网考试系统产品手册

本文档面向考官、考务人员和现场运维人员。读完后，应能独立完成一场局域网考试的考前准备、考中组织、考后导出与清场。

## 一、产品介绍

局域网考试系统是一套用于机房、教室和内网培训场景的 Web 考试系统。系统采用“管理机兼服务器”的方式运行：考官在管理机本机打开管理台，学员通过同一局域网访问考试端。

系统默认适合离线考场使用。考官管理台免登录，但仅允许在管理机本机访问；学员端通过名单校验登录，使用姓名和身份证号进入考试。

系统主要支持：

- 名单管理：按“姓名 / 单位 / 身份证号”导入学员名单。
- 客观题考试：通过 Excel 模板导入单选、多选、判断题，系统自动判分。
- 操作题考试：通过 Word 试卷导入操作题，按【标准答案】（分值）标记空位，系统自动判分；可附带表格或压缩包附件；学员可上传或粘贴截图作为佐证。
- 混合考试：同一场考试可同时包含客观题和操作题，也可只启用其中一种。
- 座位表：系统按考试随机分配座位，可在设置中控制是否展示。
- 考后整理：支持导出成绩明细和操作题截图 ZIP。
- 数据清场：可在下一场考试前清除当前考官账号下的考试、题库、名单、答卷和相关上传文件。

## 二、应用场景

本系统适合以下场景：

- 机房集中考试：学校、培训机构或单位在同一局域网内组织统一考试。
- 离线考场考试：考场不允许访问外网，所有考试材料、运行环境和数据均在管理机本地完成。
- 上机操作考核：需要学员查看 Word 题目、填写答案、上传截图佐证的技能测试。
- 客观题测验：需要快速导入题库、自动判分并导出成绩的笔试类考试。
- 混合能力考核：同一场考试中同时考查理论知识和实际操作能力。
- 小规模补考或内部培训测评：考务人员可快速导入名单和题库，按场次组织考试。

## 三、使用指南

### 1. 考前准备

1. 在管理机上启动“局域网考试系统”托盘程序。
2. 在管理机本机浏览器打开管理台：`http://127.0.0.1:5180/admin`。
3. 进入“设置”，确认是否展示座位表。
4. 如管理机保留了上一场考试数据，先确认已完成导出，再执行“清除全部数据”。
5. 准备本场考试材料：
   - 学员名单：使用官方名单模板，字段为“姓名 / 单位 / 身份证号”。
   - 客观题：使用官方题库模板。
   - 操作题：使用 Word 试卷；空位格式为【标准答案】（分值），例如【北京|北平】（2分）；多个可接受答案用 `|` 分隔。

### 2. 导入资源

在管理台依次完成：

1. 进入“名单管理”，导入本场学员名单。
2. 进入“题库管理”的“客观题”标签，导入客观题题库。
3. 进入“题库管理”的“操作题”标签，导入 Word 操作题试卷；如有素材、表格或压缩包，可作为附件一并上传。
4. 导入成功后，检查系统显示的导入数量和批次信息。

导入失败时，通常是模板列名、文件格式、空位标记或必填内容不符合要求。请按页面提示修正文件后重新导入。

### 3. 创建考试

1. 进入“考试管理”，点击新建考试。
2. 填写考试名称。
3. 选择考试内容：
   - 只考客观题：勾选客观题并选择题库批次。
   - 只考操作题：勾选操作题并选择操作题批次。
   - 混合考试：同时勾选客观题和操作题，并分别选择批次。
4. 选择本场名单批次。
5. 设置计划开始时间和计划结束时间。
6. 保存后进入考试详情页。

考试刚创建时为草稿状态。确认题库、名单和时间无误后，点击“开始考试”，名单内学员才能进入答题。

### 4. 学员参加考试

1. 考官将学员地址告知考场，格式为：`http://<管理机局域网IP>:5180/exam/login`。
2. 学员在浏览器打开学员端。
3. 输入姓名和身份证号登录。
4. 若考试尚未开始，学员进入等待页；考试开始后进入答题页。
5. 学员按页面完成作答：
   - 客观题：选择答案，系统自动保存。
   - 操作题：在页面查看 Word 试卷内容，在空位处填写答案；如需证明操作过程，可上传或粘贴截图。
6. 学员点击“提交试卷”交卷。若仍有未作答内容，系统会提示缺项，学员确认后仍可提交，未答部分按 0 分计。

作答过程中系统会自动保存，并定期同步进度，便于断网恢复或换机续考。到达计划结束时间后，系统会自动收卷并结束考试。

### 5. 考中管理

考官可在考试详情页查看考试状态、名单和座位表。若设置中关闭座位表，学员端和管理端考试详情均不展示座位信息。

如需提前结束考试，可在考试详情页点击“结束考试”。系统会先对未交卷学员执行强制交卷，再结束考试。

### 6. 考后导出与清场

考试结束后，在考试详情页执行：

1. 点击“导出成绩与明细”，下载成绩 Excel。文件包含成绩汇总、客观题答题明细和操作题明细。
2. 若考试包含操作题，可点击“导出操作题截图”，下载已交卷学员的截图 ZIP。
3. 确认成绩和材料已妥善保存后，若下一场考试不复用本场数据，可进入“设置”执行“清除全部数据”。

“清除全部数据”不可恢复，请只在确认导出完成后使用。

## 四、操作流程速览

| 阶段 | 考官操作 | 学员状态 | 结果 |
| --- | --- | --- | --- |
| 考前安装与启动 | 启动托盘程序，打开本机管理台 | 暂不可访问考试 | 系统服务就绪 |
| 数据准备 | 导入名单、客观题、操作题 | 暂不可访问考试 | 形成可选批次 |
| 创建考试 | 选择考试内容、名单和考试时间 | 暂不可作答 | 生成草稿考试 |
| 开始考试 | 在考试详情点击“开始考试” | 登录后进入答题或等待自动跳转 | 考试进入进行中 |
| 考中作答 | 查看状态，必要时查看座位表 | 自动保存、定期同步、可手动交卷 | 答案持续保存 |
| 到点或提前结束 | 系统自动结束，或考官点击“结束考试” | 未交卷学员被强制收卷 | 考试结束，禁止继续作答 |
| 考后整理 | 导出成绩，导出操作题截图 ZIP | 已交卷学员可只读查看 | 获得考后归档材料 |
| 下一场前 | 确认导出后清除全部数据 | 需重新登录下一场 | 清空本场业务数据 |

## 五、使用注意事项

- 管理台只能在管理机本机打开，推荐地址为 `http://127.0.0.1:5180/admin`。学员机访问管理台应被拦截或提示在本机打开。
- 学员端只需要访问 `http://<管理机局域网IP>:5180/exam/login`，不要让学员使用管理台地址。
- 现场建议保持管理机、电源和局域网稳定。考试过程中不要退出托盘程序，不要关闭管理机。
- 关闭托盘窗口通常只是最小化到托盘；只有通过托盘菜单退出系统，才会停止服务。
- 考试材料应提前离线准备好。考场内不要依赖外网下载模板、运行库或浏览器插件。
- 学员姓名和身份证号必须与导入名单一致。除首尾空格外，姓名和身份证号不一致会导致登录失败。
- 操作题 Word 中的空位必须使用【标准答案】（分值）格式；多个可接受答案使用 `|` 分隔。
- 操作题截图仅作为佐证材料，不参与自动评分。自动评分以题目空位答案为准。
- 到达计划结束时间后，系统会自动交卷并结束考试。请在创建考试时认真核对结束时间。
- “清除全部数据”会删除考试、题库、名单、答卷和上传文件，但保留设置项；执行前务必先导出成绩和材料。
- 默认免登录数据归属于本机本场管理账号；旧账号数据不会自动迁移，升级或切换环境后可能需要重新导入。
- HTTP 仅适合机房内网使用，不应暴露到公网。

## 六、场景问题Q&A

**Q1：学员机打不开考试页面怎么办？**  
A：先确认管理机服务已启动，并能在本机打开管理台。再检查学员访问的是否是管理机局域网 IP，而不是 `127.0.0.1`。如果仍无法访问，检查管理机防火墙是否放行 5180 端口，以及管理机和学员机是否在同一局域网。

**Q2：学员登录失败怎么办？**  
A：检查姓名和身份证号是否与导入名单完全一致。重点核对身份证号中的字母、空格、全角半角和复制粘贴错误。必要时在名单详情中搜索该学员并修正。

**Q3：学员登录后一直在等待页怎么办？**  
A：通常是考试尚未开始。考官进入考试详情页，确认考试是否已点击“开始考试”，并确认该学员属于本场考试选择的名单批次。

**Q4：考试时间到了，学员还没有手动交卷怎么办？**  
A：系统会按计划结束时间自动收卷。已保存的答案会参与计分，未答部分按 0 分计。考官也可以在考试详情页提前点击“结束考试”。

**Q5：学员交卷时提示还有未答内容，可以提交吗？**  
A：可以。系统会列出缺项，学员确认后仍可交卷，未作答的客观题或操作题按 0 分计。

**Q6：操作题导入失败怎么办？**  
A：检查 Word 文件是否为支持格式，是否按【标准答案】（分值）标记空位，分值是否填写清楚，多个可接受答案是否用 `|` 分隔。修正后重新导入。

**Q7：操作题截图上传失败怎么办？**  
A：确认截图格式为 PNG、JPG/JPEG 或 WebP，数量未超过每空限制，单张文件大小符合系统限制。也可尝试重新截图后再上传。

**Q8：考后为什么导出的操作题截图 ZIP 里没有某个学员？**  
A：截图 ZIP 只包含已交卷学员实际上传或粘贴过的截图。若学员未交卷，或该学员没有上传截图，ZIP 中不会出现对应截图。

**Q9：能不能在学员机上打开管理台？**  
A：不可以。管理台设计为只在管理机本机使用。学员机应只访问考试端，避免误操作或泄露管理权限。

**Q10：下一场考试前是否必须清除全部数据？**  
A：如果下一场需要全新名单和题库，建议在确认本场导出完成后清除全部数据。若下一场复用同一批题库或名单，可不清除，但创建考试时要仔细选择正确批次。
